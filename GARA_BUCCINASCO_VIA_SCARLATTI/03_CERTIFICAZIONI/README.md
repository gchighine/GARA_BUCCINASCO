# 03 - Certificazioni

Questa cartella contiene tutte le certificazioni aziendali richieste per la partecipazione alla gara.

## Certificazioni da Inserire

### Certificazioni ISO

- **ISO_9001_AP_Srl.pdf**: Certificazione del Sistema di Gestione per la Qualità.
- **ISO_14001_AP_Srl.pdf**: Certificazione del Sistema di Gestione Ambientale.

### Iscrizioni Albo Gestori Ambientali

- **Albo_Gestori_Cat4.pdf**: Categoria 4 - Raccolta e trasporto di rifiuti speciali non pericolosi.
- **Albo_Gestori_Cat5.pdf**: Categoria 5 - Raccolta e trasporto di rifiuti speciali pericolosi.
- **Albo_Gestori_Cat9.pdf**: Categoria 9 - Bonifica di siti e beni contenenti amianto.
- **Albo_Gestori_Cat10a.pdf**: Categoria 10a - Bonifica di beni contenenti amianto.

## Verifica Validità

Prima di caricare le certificazioni, verificare che:

1. **Siano in corso di validità**: Controllare le date di scadenza.
2. **Siano complete**: Assicurarsi che tutti i documenti richiesti dal disciplinare siano presenti.
3. **Siano leggibili**: I file PDF devono essere di buona qualità e facilmente leggibili.

## Note

Le certificazioni sono documenti fondamentali per la partecipazione alla gara. La mancanza anche di una sola certificazione richiesta può comportare l'esclusione dalla gara.
